1. Makefile, який дозволяє зібрати код на різних платформах та архітектурах. Наприклад, якщо зібрати код для Linux, запускаємо команду "make linux", яка збере код для Linux. Те саме повинно бути зроблено для arm, macOS та Windows.
2. Також для автоматизації тестів, створено Dockerfile, який дозволить запустити тестовий набір на різних платформах та архітектурах. Отримуємо Docker-контейнер із бінарним файлом.
3. Використовуєтся альтернативний container registry: https://quay.io/repository/projectquay/golang?tab=info.
- Makefile: clean повинен містити видалення новоствореного образу docker rmi <IMAGE_TAG>
- Dockerfile: базовий образ quay.io/projectquay/golang:1.20
___________________________________________________________________________________________________________________
# Збірка коду для Linux
make linux
# Збірка коду для ARM
make arm
# Збірка коду для macOS
make macos
# Збірка коду для Windows
make windows
# Очищення згенерованих файлів / Видалення Docker-образу
make clean
# Створення Docker-образу
make image
